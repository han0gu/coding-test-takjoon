# 짝지어 제거하기 피드백

## 문제 조건 요약

- 같은 알파벳이 2개 붙어 있는 짝을 찾아 제거하고, 이 과정을 반복해 문자열을 모두 제거할 수 있으면 1을 반환한다.
- 모두 제거할 수 없으면 0을 반환한다.
- 문자열 길이는 1 이상 1,000,000 이하이고, 문자열은 모두 소문자로 이루어진다.

## Gemini 리뷰 요약

Gemini는 현재 스택 풀이의 방향은 유지하되, 조건문을 더 단순하게 정리할 수 있다고 피드백했다.

기존 코드는 스택이 비어 있는 경우를 먼저 처리하고 `continue`로 다음 반복으로 넘어간 뒤, 다시 `head = stack[-1]` 임시 변수를 만들어 현재 문자와 비교했다.

```python
if not stack:
    stack.append(c)
    continue

head = stack[-1]
if head == c:
    stack.pop()
else:
    stack.append(c)
```

이 구조도 동작은 맞지만, 같은 판단을 하나의 조건으로 합칠 수 있다.

## 반영한 수정

스택에 값이 있고 마지막 문자가 현재 문자와 같으면 제거하고, 그렇지 않으면 현재 문자를 스택에 넣도록 정리했다.

```python
if stack and stack[-1] == c:
    stack.pop()
else:
    stack.append(c)
```

이렇게 쓰면 다음 장점이 있다.

- 스택이 비어 있을 때도 `else`에서 자연스럽게 문자를 추가한다.
- `continue`가 없어져 반복 흐름이 단순해진다.
- `head` 임시 변수를 만들지 않아도 비교 의도가 바로 드러난다.

## 정리

이 문제의 핵심은 인접한 같은 문자를 만나면 바로 제거하고, 제거 후 새롭게 맞닿는 문자도 다시 비교해야 한다는 점이다.

스택은 이 흐름과 잘 맞는다. 마지막으로 남아 있는 문자만 확인하면 되므로 전체 문자열을 여러 번 다시 훑지 않아도 되고, 문자열 길이가 최대 1,000,000이어도 한 번의 순회로 처리할 수 있다.
