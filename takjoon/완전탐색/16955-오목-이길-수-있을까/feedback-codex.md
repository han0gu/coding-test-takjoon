# 방향 배열로 중복 줄이기

현재 `main.py`는 세로, 가로, 대각선 방향을 각각 `check_0`, `check_45`, `check_90`, `check_135` 함수로 나누어 확인하고 있습니다.

이 방식도 정답을 만들 수 있지만, 네 함수의 구조가 거의 같습니다.

- 시작 위치를 정한다.
- 특정 방향으로 5칸을 본다.
- 그 5칸 안에 `O`가 없는지 확인한다.
- 빈칸 `.`이 1개 이하인지 확인한다.

따라서 다른 부분은 "어느 방향으로 움직일 것인가"뿐입니다. 이때 방향 배열을 쓰면 중복을 줄일 수 있습니다.

## 방향 배열이란?

2차원 배열에서 위치는 보통 `(row, col)`로 표현합니다.

- `row`가 1 증가하면 아래로 이동합니다.
- `col`이 1 증가하면 오른쪽으로 이동합니다.
- `row`가 1 감소하면 위로 이동합니다.

그래서 방향 하나를 `(dr, dc)`로 표현할 수 있습니다.

```python
directions = [
    (1, 0),   # 세로 아래
    (0, 1),   # 가로 오른쪽
    (1, 1),   # 오른쪽 아래 대각선
    (-1, 1),  # 오른쪽 위 대각선
]
```

예를 들어 현재 위치가 `(row, col)`이고 방향이 `(1, 0)`이면 다음 칸들은 다음과 같습니다.

```python
(row + 0, col)
(row + 1, col)
(row + 2, col)
(row + 3, col)
(row + 4, col)
```

즉, 세로 방향으로 5칸을 확인합니다.

방향이 `(0, 1)`이면 다음과 같습니다.

```python
(row, col + 0)
(row, col + 1)
(row, col + 2)
(row, col + 3)
(row, col + 4)
```

즉, 가로 방향으로 5칸을 확인합니다.

## 현재 코드와의 대응

현재 코드의 함수들은 방향 배열로 다음처럼 바꿔 생각할 수 있습니다.

| 현재 함수 | 의미 | 방향 |
| --- | --- | --- |
| `check_0` | 세로 아래 | `(1, 0)` |
| `check_90` | 가로 오른쪽 | `(0, 1)` |
| `check_135` | 오른쪽 아래 대각선 | `(1, 1)` |
| `check_45` | 오른쪽 위 대각선 | `(-1, 1)` |

즉, 네 개의 함수를 따로 둘 필요 없이 `directions`를 순회하면서 같은 로직을 반복할 수 있습니다.

## 적용 예시

`main.py`를 방향 배열 방식으로 정리하면 아래처럼 쓸 수 있습니다.

```python
N = 10
board = [input() for _ in range(N)]

directions = [
    (1, 0),
    (0, 1),
    (1, 1),
    (-1, 1),
]

answer = 0

for row in range(N):
    for col in range(N):
        for dr, dc in directions:
            end_row = row + dr * 4
            end_col = col + dc * 4

            if 0 <= end_row < N and 0 <= end_col < N:
                candidates = [
                    board[row + dr * i][col + dc * i]
                    for i in range(5)
                ]

                if 'O' not in candidates and candidates.count('.') <= 1:
                    answer = 1

print(answer)
```

## 왜 끝 위치를 먼저 확인할까?

5칸을 확인하려면 시작 칸뿐만 아니라 마지막 칸도 바둑판 안에 있어야 합니다.

예를 들어 `(row, col)`에서 `(dr, dc)` 방향으로 5칸을 보면 마지막 위치는 다음과 같습니다.

```python
end_row = row + dr * 4
end_col = col + dc * 4
```

이 마지막 위치가 보드 안에 있는지만 확인하면, 중간 1칸, 2칸, 3칸도 모두 보드 안에 있습니다.

그래서 아래 조건 하나로 범위를 확인할 수 있습니다.

```python
if 0 <= end_row < N and 0 <= end_col < N:
```

이렇게 쓰면 방향별로 다른 범위 조건을 직접 작성하지 않아도 됩니다.

## `count('.')` 사용

현재 코드에서는 빈칸 개수를 셀 때 `filter`를 사용하고 있습니다.

```python
len(list(filter(lambda x: x == '.', candidates))) <= 1
```

이 경우에는 `count`를 쓰는 편이 더 단순합니다.

```python
candidates.count('.') <= 1
```

이 문제에서는 `candidates`가 항상 길이 5인 작은 리스트이므로 성능 차이는 신경 쓰지 않아도 됩니다. 읽기 쉬운 쪽을 선택하는 것이 좋습니다.

## 정리

방향 배열은 "같은 동작을 여러 방향에 대해 반복할 때" 유용합니다.

이번 문제에서는 세로, 가로, 대각선 확인 로직이 모두 같고 방향만 다르기 때문에 방향 배열을 적용하기 좋은 예시입니다.
